# Manjaro Edition of the Flat Remix GTK-Theme Darkest with no border
This is an alternative theme version of the Flat-Remix-GTK-Green-Darkest-Solid-NoBorder by daniruiz (see [https://github.com/daniruiz/Flat-Remix-GTK](https://github.com/daniruiz/Flat-Remix-GTK) and [https://drasite.com/flat-remix-gtk](https://drasite.com/flat-remix-gtk)) made for manjaro users. 

## Changes made to original files
* Changed the blue circle to manjaro green 

## Installation
* Copy the folder to ~/.themes/  
* Reboot
* Select theme in gnome tweak
